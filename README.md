## 📜 The Nomencraft Doctrine  

Welcome, Initiate. You stand at the threshold of **Artis Nomen** — the ancient and entirely made‑up discipline devoted to the shaping of filenames with clarity, intention, and just a hint of unnecessary ceremony.

### ✨ Our Tenets

- 🧵 **Clarity Above Chaos**  
  A filename should not resemble a ransom note or a cryptic prophecy.

- 🙏 **Patterns Are Sacred**  
  Through structured transformation, enlightenment (and correct sorting) is achieved.

- ☮️ **Determinism Is Peace**  
  Surprises belong in birthday parties, not file operations.

- 🔨 **Craft Over Force**  
  We do not bludgeon filenames into shape.  
  We *coax* them. Gently. Like civilized people.

### 🕯️ The Path of the Renamer

All who walk the path begin with a single file.  
Then another.  
Then an entire directory of horrors someone named “final_FINAL_reallyfinal2”. 🤦  
Through Nomencraft, these burdens are lifted.

### 🧂 A Note on Humility  

Nomencraft is a tool of great power and zero chill.  
Use it wisely. Or recklessly. We’re not your supervisor. 🤷

## What This Is  

Nomencraft began as a general‑purpose, pattern‑powered renaming tool — the kind you build when filenames start looking like they were generated by a caffeinated raccoon. ☕🦝☕  

Then it took a detour into handling a very specific set of lab files ⚗️ with very specific chaos.  
Right now, that’s the proving ground. After that? We’ll see where the craft leads. 🐾

## Features  

- ⚙️ Pattern‑driven renaming with predictable results  
- 🔄 Deterministic pipelines (because surprises belong in birthday parties, not file operations)  
- 👓 A philosophy that values clarity over cleverness  
- 🗺️ A roadmap that starts with lab files and expands outward
